readability-api
===============

### Example

```bash
$ curl -s https://readability-api.herokuapp.com/?fetch_url=https://www.twreporter.org/a/opinion-scandinavia-refugee 
```

### Install

```bash
$ git clone https://github.com/poying/readability-api.git
$ cd readability-api
$ npm install
$ POER=3000 npm start
```
